# Auto Key Presser

A simple Windows app that automatically presses a key or keyboard combination at the interval you choose.

## Download

Download [`AutoKeyPresser.exe`](dist/AutoKeyPresser.exe), place it anywhere you like, and open it. The app is portable and does not need to be installed.

## How to use it

### 1. Choose a key

Click **Select key**, then press the key you want the app to repeat.

You can also choose combinations such as:

- `Shift + F`
- `Shift + 2`
- `Ctrl + Q`
- `Ctrl + 1`
- `Alt + 1`

Press and hold the modifier, then press the main key. If you release `Shift`, `Ctrl`, or `Alt` before choosing the main key, it is removed from the combination.

Click **Cancel** to keep the previously selected key.

### 2. Set the interval

Enter how often the key should be pressed and choose:

- **milliseconds** for short intervals,
- **seconds** for longer intervals.

The minimum interval is 50 milliseconds.

### 3. Optional random deviation

Enable **Random deviation** to make the delay slightly different each time.

For example, an interval of `1000 ms` with a deviation of `100 ms` produces delays between `900 ms` and `1100 ms`.

### 4. Optional start delay

Use **Start delay** to give yourself time to switch to another window before pressing begins.

### 5. Choose when to stop

The **Stop mode** menu provides three options:

- **No limit** ŌĆö runs until you press **Stop**.
- **After a number of presses** ŌĆö stops after the selected number of key presses.
- **After a set time** ŌĆö stops after the selected number of milliseconds or seconds.

The time limit must be at least as long as one interval. If you only need a single press, choose **After a number of presses** and enter `1`.

### 6. Start

Click **Start**, then switch to the window that should receive the key presses. The app always sends input to the currently active window.

While running, you can see:

- the current status,
- the total number of completed presses,
- the countdown to the next press.

Click **Stop** at any time to finish.

## Stop when another key is pressed

When **Stop when another key is pressed** is enabled, pressing any physical key outside the selected key or combination immediately stops the app. Key presses generated by Auto Key Presser do not trigger this option.

## Saved settings

Your most recent settings are saved automatically and restored the next time you open the app.

## Important notes

- Key presses are sent to whichever window is currently active.
- Some games or protected applications may ignore automatic input.
- An app running as administrator may not accept input from Auto Key Presser unless Auto Key Presser is also started as administrator.
